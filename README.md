Identificador de Pessoas em Python

📌 Descrição

Este é um programa simples desenvolvido em Python que identifica uma pessoa a partir do nome informado no código e retorna qual é a relação dessa pessoa com o usuário.

O programa utiliza estruturas condicionais (if, elif, else) para verificar o nome e exibir uma mensagem correspondente.

🚀 Tecnologias utilizadas
	•	Python

🧠 Lógica do programa

O programa compara o valor da variável Nome com alguns nomes pré-definidos:
	•	Vitória Rosa → Amor da vida
	•	Maria Libencia → Mãe
	•	Mayla → Filhota
	•	Ana Julia → Sobrinha

Caso o nome não esteja na lista, o programa retorna:
Quem é essa ?

▶️ Como executar o programa
	1.	Instale o Python no seu computador
	2.	Clone o repositório
	3.	Execute o arquivo:

python identificador.py
